1-D Arrays

Arrays are a type of data structure that can hold a specific type of data. This is very useful when trying to have a collection of data of the same data type for example to store all the names of students in a class an array is very useful instead of using multiple string variables to store this value. In technical terms, an array is a block of memory locations that hold values of the same data type and an individual location is called an element. An element can be accessed through the index, in arrays the first element is index 0 and the nth element is n-1 and can be accessed like a[n-1]. In Java an array is a fixed length, therefore, it canâ€™t be changed however you can create a new array and set the array equal to the new array to change its length as shown. An index can be out of bounds if it is less than 0 or greater than the array length. An array is an object itself therefore it acts similarly to a string when passed as a reference or when doing something else.

JAVA
  public class ArrayExample {

    public static void main(String[] args) {
        
        // Using array literal to declare and initialize an array
        String[] fruits = {"Apple", "Banana", "Orange"}; 

        // Declaring an integer array
        int[] arr = new int[5]; //array of length 5

        // Initializing an array
        arr[0] = 1;
        arr[1] = 2;
        arr[2] = 3;
        arr[3] = 4;
        arr[4] = 5;

        // Accessing elements of an array using index
        System.out.println(arr[2]); // Output: 3
        System.out.println(fruits[1]); // Output: Banana

        // Looping through an array
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + " ");
        }
        // Output: 1 2 3 4 5

        // Using for-each loop to loop through an array
        for (String fruit : fruits) {
            System.out.print(fruit + " ");
        }
        // Output: Apple Banana Orange
    }
  }

2-D Arrays

2D arrays can be visualized as a table with rows and columns while still holding the same type of data. It is basically a 1d array of 1d arrays with rows. Java allows ragged arrays therefore it is important to use .length instead of the main row. To loop through a 2D array the use of nested loops is necessary as it loops through the rows and columns. Each loop is a shorter loop where a variable is set and is looped through an object and instead of accessing through the index it can be accessed directly.

JAVA
  public class Array2DExample {

    public static void main(String[] args) {
        
      //Declaring a 2D array
        int[][] matrix = new int[2][2];
        
        // Initializing the elements of the 2D array
        matrix[0][0] = 1;
        matrix[0][1] = 2;
        matrix[1][0] = 3;
        matrix[1][1] = 4;

        //Declaring and Initializing a 2D array
        int[][] arr = {{10,11}{12,13}};
        
        // Accessing elements of the 2D array
        System.out.println(matrix[0][0]); // Output: 1
        System.out.println(matrix[1][1]); // Output: 4
        
        // Looping through the 2D array
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr[i].length; j++) {
                System.out.print(arr[i][j] + " ");
            }
            System.out.println();
        }
    }
  }

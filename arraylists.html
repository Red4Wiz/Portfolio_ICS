ArrayLists

Arrays list implements a list using an array and can only hold objects not elements of a primitive data type. Unlike arrays, array lists can change their lengths so they can decrease or increase instead of having a fixed length. Uses get and set methods to change a value instead of an array that you just had to access using an index. Autoboxing is converting a primitive data type into an object for example an int to an Integer object, and this is what happens every time you set it in an array list. Autounboxing is when converting an Integer object to an int data type. It is useful to use for each loop when working with an ArrayList. Therefore the ArrayList is basically a flexible version of an array that can only hold objects instead of primitive data types.

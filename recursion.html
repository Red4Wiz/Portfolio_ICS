Recursion

Recursion is a programming technique where a method invokes itself to solve a problem by breaking it down into smaller, simpler subproblems (similar to divide and conquer where you split a big problem into different subproblems). All recursive methods contain two components the base case and the recursive case.

Recursive Case:

The recursive case keeps breaking the main problem into smaller sub problems to solve the question. This contains the recursion as it involves making one or more recursive calls to the method itself with a modified input. Each recursive call is intended to handle a smaller input, slowly moving to the base case.

Base Case:

The base case is the condition that specifies when the recursive method should stop calling itself and return a value. It contains the simplest version of the problem that can be solved directly without further recursion. The base case ensures that the recursive calls eventually terminate and a base case is required in all recursive methods so the method can terminate.

JAVA
  public static int factorial(int n) {
    // Base case: 0! and 1! = 1
    if (n == 0 || n == 1) {
        return 1;
    } else {
        // Recursive case: n! = n * (n-1)!
        return n * factorial(n - 1);
    }
  }

Recursion VS Iteration

A recursive method is a function that calls itself to solve the problem by breaking it down into smaller subproblems. It relies on the concept of a base case, which determines when the recursion should stop. Recursive methods are often concise and elegant, especially for problems with inherent recursive structures. On the other hand, an iterative method uses loops and variables to repeatedly instructions until the certain condition is met. Iterative methods are generally more efficient in terms of memory usage and can be easier to understand for some problems. It is especially useful for problems where you know the number of times to execute the instruction as the iterative method can be more efficiently controlled. For example we will provide an example to solidify our knowledge, to find the sum of all the numbers from 1 to n here are two different ways to achieve the task.

Recursive Solution:

JAVA
  public static int sumRecursive(int n) {
      if (n == 1) {
          return 1;
      } else {
          return n + sumRecursive(n - 1);
      }
  }

Iterative Solution:

JAVA
  public static int sumIterative(int n) {
      int sum = 0;
      for (int i = 1; i <= n; i++) {
          sum += i;
      }
      return sum;
  }
